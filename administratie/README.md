# Administratie

De map `administratie/` bundelt alle documenten met juridische, contractuele, boekhoudkundige of conformiteitswaarde.

| Submap | Gebruik |
| --- | --- |
| `algemene_voorwaarden/` | Voorwaarden die samen met offertes of opdrachten meegestuurd worden |
| `contracten/` | Onderhoudscontracten, raamovereenkomsten, serviceafspraken |
| `certificaten/` | Bevoegdheden, opleidingen en relevante bedrijfsdocumenten |
| `attesten/` | Opleverattesten, keuringsverslagen, conformiteitsattesten |
| `facturen/` | Uitgaande facturen en creditnota's |
| `offertes/` | Niet-klantgebonden of administratief gearchiveerde offertes |

Administratieve documenten volgen de centrale naamgevingsregels en vallen mee onder Git-versiebeheer voor historiek en traceerbaarheid.

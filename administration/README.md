# Administratie

De map `administratie/` bevat alle bedrijfsdocumenten die juridisch, contractueel of boekhoudkundig relevant zijn.

| Submap | Doel |
| --- | --- |
| `algemene_voorwaarden/` | Officiële voorwaarden die mee kunnen met offertes en opdrachten |
| `contracten/` | Servicecontracten, onderhoudsovereenkomsten, raamafspraken |
| `certificaten/` | Bevoegdheden, kwalificaties en certificaten |
| `attesten/` | Opleverattesten, keuringsattesten, conformiteitsdocumenten |
| `facturen/` | Uitgaande facturen en creditnota's |
| `offertes/` | Archiefkopieën van algemene of niet-klantgebonden offertes |

Administratieve documenten volgen dezelfde datum- en naamgevingslogica als technische documenten. Voor facturen gelden wettelijke bewaartermijnen van minstens 7 jaar. Contracten worden bewaard voor de looptijd plus 5 jaar. Attesten en certificaten blijven permanent raadpleegbaar zolang ze relevant zijn voor werking of aansprakelijkheid.

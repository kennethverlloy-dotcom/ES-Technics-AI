# Persoonlijke Voorkeuren

De map `persoonlijk/` bevat de specifieke voorkeuren, werkmethodes en schrijfstijl van de zaakvoerder van ES-Technics BV. 

Waar `es-technics/` de formele bedrijfsstandaarden bevat, bepaalt deze map de *toon* en de *praktische aanpak*. De AI leest deze map bij elke taak om ervoor te zorgen dat de output niet klinkt als een generieke robot, maar als een ervaren medewerker van ES-Technics.

## Inhoud

| Map | Doel |
| --- | --- |
| `leveranciers/` | Voorkeursmerken per categorie (bijv. WAGO, Belimo, Systemair) en de regel om bestaande kanalen te respecteren. |
| `opmaak/` | Voorkeuren voor documentstructuur (tabellen voor materialen, chronologische opbouw voor diagnoses). |
| `schrijfstijl/` | Instructies voor een zakelijk Vlaams register: kort, feitelijk, professioneel. |
| `standaard_antwoorden/` | Herbruikbare e-mailstructuren voor prijsaanvragen, afspraken en klachten. |
| `werkmethodes/` | Praktische richtlijnen voor veiligheid, back-ups en documentatie. |

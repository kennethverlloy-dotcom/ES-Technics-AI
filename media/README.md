# Media

De map `media/` bewaart ruwe bestanden die gekoppeld kunnen worden aan klanten, projecten, rapporten en dossiers.

| Submap | Inhoud |
| --- | --- |
| `fotos/` | Foto's van installaties, defecten, labels, displays |
| `videos/` | Bewegende beelden van storingen of testprocedures |
| `schemas/` | Exporten van hydraulische, elektrische of P&ID-schema's |
| `pdf/` | Externe handleidingen, rapporten en fiches |
| `technische_tekeningen/` | Plannen, technische details, layouts |
| `cad/` | CAD-bronbestanden |
| `dwg/` | DWG-uitwisselingsbestanden |

Gebruik de naamgevingsconventie `JJJJ-MM-DD_klant-of-project_omschrijving.extensie`. In klant- en projectdossiers wordt bij voorkeur met relatieve paden of duidelijke verwijzingen naar mediabestanden gewerkt. Indien de repository later zeer groot wordt, is Git LFS aanbevolen voor zware bestanden.

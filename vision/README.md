# Vision: AI-Fotoanalyse Richtlijnen

Deze map bevat de richtlijnen voor het analyseren van foto's en schema's door de AI-medewerker.

## Werkwijze bij Visuele Analyse

Wanneer de gebruiker een foto of schema uploadt en vraagt om een analyse, volgt de AI deze gestructureerde aanpak:

1.  **Identificatie:** Wat staat er op de foto? (Type installatie, component, schema-soort).
2.  **Observatie:** Wat is de huidige staat? (Zichtbare schade, instellingen op een display, aansluitingen).
3.  **Vergelijking:** Komt dit overeen met de standaarden en best practices van ES-Technics?
4.  **Diagnose/Conclusie:** Wat is het waarschijnlijke probleem of de status?
5.  **Aanbeveling:** Wat moet er gebeuren? (Vervanging, reiniging, meting).

## Categorieën

*   `installaties/`: Algemene opstellingen van luchtgroepen, chillers, ketels.
*   `schemas/`: P&ID (Piping and Instrumentation Diagram) en elektrische schema's.
*   `elektrische_borden/`: Kastenbouw, bekabeling, componenten (PLC, automaten).
*   `chillers/`: Specifieke koeltechnische componenten (compressoren, condensors).
*   `luchtgroepen/`: Filters, ventilatoren, kleppenregisters.
*   `storingen/`: Typische visuele defecten (lekkages, verbrande contacten, vervuiling).
